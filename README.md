# Gauri – ERP Management Website

## Overview

Gauri is a responsive ERP management website developed to streamline business operations through integrated modules for Human Resources, Sales, Inventory, Purchase, and Accounting. The platform focuses on providing an intuitive user experience, modern design, and efficient business management solutions.

## Features

* Responsive design optimized for desktop, tablet, and mobile devices
* ERP modules for:

  * Human Resources (HR)
  * Sales Management
  * Inventory Management
  * Purchase Management
  * Accounting
* AI-powered chatbot for real-time user assistance
* Cross-browser compatibility
* Reusable and scalable UI components
* Modern and user-friendly interface

## Tech Stack
